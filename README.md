# EDA_Project

This project analyzes the Netflix Movies and TV Shows dataset using Python.

## Objective
To clean the dataset, perform exploratory data analysis, create visualizations, identify trends, and generate meaningful business insights.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Project Tasks
- Data Cleaning
- Exploratory Data Analysis (EDA)
- 6 Visualizations
- Business Insights Generation

## Key Findings
1. Netflix has more Movies than TV Shows.
2. United States contributes the highest content.
3. TV-MA is the most common content rating.
4. Netflix content increased rapidly in recent years.
5. Drama and International Movies are highly popular.
